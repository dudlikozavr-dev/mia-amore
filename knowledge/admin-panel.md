# Веб-панель администратора — как зайти и добавить товар

## Адрес панели

**Продакшн:** https://mia-amore-production.up.railway.app/admin

**Локально (для разработки):**
1. Запустить бэкенд:
   ```bash
   cd backend
   venv_312\Scripts\activate
   uvicorn main:app --reload --port 8000
   ```
2. Открыть: http://localhost:8000/admin

## Вход

При открытии панели появляется окно «Введите токен».

**Токен:** значение переменной `ADMIN_API_TOKEN` из `.env` (локально) или из Railway Variables (продакшн).

- Посмотреть локально: файл [backend/.env](backend/.env) → строка `ADMIN_API_TOKEN=...`
- Посмотреть в проде: Railway → проект `mia-amore` → Variables → `ADMIN_API_TOKEN`

Токен сохраняется в localStorage браузера — вводится один раз на устройство.

## Разделы панели

| Раздел      | Что делает                                         |
|-------------|----------------------------------------------------|
| Дашборд     | Статистика: заказы, покупатели, выручка            |
| **Товары**  | Добавить / редактировать / удалить товар и фото    |
| Заказы      | Список заказов, смена статуса                      |
| Покупатели  | База покупателей (Telegram ID, заказы)             |
| Рассылка    | Отправить сообщение всем покупателям               |

## Добавление нового товара (пошагово)

1. Открыть панель → раздел **«Товары»**.
2. Нажать **«+ Добавить товар»**.
3. Заполнить поля:
   - **Название** (например, «Халат Bella чёрный»)
   - **Категория** — выбрать из списка (робы/пижамы/комплекты). Если нужной нет — сначала создать в API/БД.
   - **Материал / Материал (подпись)** — `silk` / «Шёлк 100%, 19 momme»
   - **Цена** и **Старая цена** (в рублях, целое число)
   - **Бейдж** — `hit` / `new` / пусто
   - **Остаток** (stock)
   - **Размеры** — через запятую: `XS, S, M, L, XL`
   - **Неактивные размеры** (`disabled_sizes`) — те, которых нет в наличии
   - **Цвета** — JSON: `[{"name":"Чёрный","hex":"#1C1C1E"}]`
   - **Описание** и **Уход**
4. Сохранить — товар появится в списке.
5. Нажать на товар → раздел **«Фото»** → загрузить 3–4 изображения (WebP/JPG, 600×800).
   - Первое фото становится главным на карточке.
   - Можно добавить фото таблицы размеров (тип `size_chart`).
6. Убедиться, что **«Активен»** = ✅ — иначе товар не появится в каталоге фронта.

## Как проверить

- Открыть бота **@sikretsweet_home_bot** → кнопка меню → новый товар должен быть в каталоге.
- Если товар не появился: проверить `is_active=true` и что у товара есть хотя бы одно фото.

## Альтернатива: через бота

В Telegram боту доступны команды:
- `/start` — меню админа
- `/orders` — последние заказы
- `/order N` — детали заказа

Добавление товаров через бота пока не реализовано — только через веб-панель.

## Troubleshooting

- **«Нет доступа» (403)** — неправильный токен. Очистить localStorage (F12 → Application → Local Storage → удалить `admin_token`), ввести заново.
- **«Админский токен не настроен» (503)** — в env не задан `ADMIN_API_TOKEN`. Задать на Railway и перезапустить.
- **Панель не открывается** — проверить `https://mia-amore-production.up.railway.app/health`. Если не отвечает — бэкенд упал, смотреть логи в Railway.
