## 1. Fashion Fabrique

**Бот:** @fashionfabrique_bot  
**Платформа:** TGShop  
**Ниша:** женская одежда, 626 000 подписчиков в соцсетях

**Что делает:**  
Полноценный Mini App магазин — каталог с категориями, карточки товаров, корзина, онлайн-оплата, интеграция с доставкой. Автоматические рассылки по базе посетителей.

**Результаты:**  
- Продажи выросли на 30% за 2 месяца после запуска
- Увеличилось количество позиций в заказе
- Вернулась аудитория, которая не хотела общаться с консультантами

**Сильные стороны:**
- Автоматизация заказов (раньше принимали вручную через консультантов)
- Рассылка всем посетителям магазина
- Каталог синхронизирован с основным складом

**Слабые стороны:**
- Шаблонный дизайн TGShop — нет уникальности бренда
- Ограниченные возможности кастомизации конструктора

**Что взять себе:**
- Автоматические уведомления по базе — работает для возврата клиентов
- Модель «канал с контентом + Mini App для покупок» — разделение ролей

**Ссылки:**  
- [Кейс на TGShop](https://tgshop.io/shops/fashion-fabrique/)

## 2. Yammy — кастомный Web App для магазина одежды

**Тип:** кастомная разработка (не конструктор)  
**Ниша:** небольшой магазин одежды

**Что делает:**  
Mini App с полным UX-процессом: mind map → бумажные прототипы → Figma → юзер-тесты. Поиск товаров, избранное, промокоды, сетка товаров с фото, корзина, оплата.

**Сильные стороны:**
- Продуманный UX: прошёл полный цикл проектирования
- Поиск товаров в шапке — редкость для Mini App
- Избранное (вишлист) — большинство конкурентов этого не имеют
- Промокоды и система скидок
- Сетка товаров с иконками быстрого добавления в корзину и избранное

**Слабые стороны:**
- Индивидуальная разработка — дороже и дольше конструктора
- Нет публичных данных о конверсии

**Что взять себе:**
- UX-подход: прототипирование до кода
- Иконки быстрых действий на карточках (корзина + сердце)
- Поиск в шапке приложения

**Ссылки:**  
- [Обзор на vc.ru](https://vc.ru/services/1600999-yammy-web-app-dlya-magazina-odezhdy-v-telegram)

## 3. G.Shop (@GHop_Bot)

**Бот:** @GHop_Bot  
**Тип:** бот с каталогом  
**Ниша:** магазин одежды

**Что делает:**  
Магазин одежды через Telegram-бот. Меню: Каталог / Корзина / Заказы / Новости / Настройки / Помощь. Карточки товаров с фото на моделях, выбор размера, кнопка «Купить».

**Сильные стороны:**
- Понятная структура меню — 6 разделов, всё на виду
- Фото на моделях, а не на вешалке
- Карточки с ключевыми параметрами (размер, цена)
- Приветственное сообщение с информацией о товарах и доставке

**Слабые стороны:**
- Формат бота (кнопки в чате), а не полноценный Mini App — ощущается устаревшим
- Нет визуальной сетки товаров — всё через сообщения
- Медленная навигация: каждый шаг — новое сообщение от бота

**Что взять себе:**
- Структура меню: Каталог / Корзина / Заказы / Новости — проверенный набор
- Приветственное сообщение с ключевой информацией при первом запуске

**Ссылки:**  
- [Обзор на botobot.ru](https://www.botobot.ru/blog/ru/magazin-odezdy-telegram/)

## 4. CASHER COLLECTION

**Бот:** @casher_collection_bot  
**Платформа:** TGShop  
**Ниша:** трендовая молодёжная одежда, собственный бренд

**Что делает:**  
Mini App магазин с каталогом по категориям, корзиной, онлайн-оплатой, интеграцией с доставкой. Коллаборации с блогерами (Амкал, Клик-клак) как источник трафика.

**Сильные стороны:**
- Telegram как основной канал продаж — не дополнение к сайту, а главная витрина
- Коллаборации с блогерами направляют трафик прямо в бота
- Эксклюзивность: товары не продаются на маркетплейсах
- Каталог с категориями, удобная навигация

**Слабые стороны:**
- Шаблонный дизайн TGShop — выглядит как все остальные магазины на этой платформе
- Нет уникальной айдентики внутри Mini App
- Ограниченная кастомизация карточек товара

**Что взять себе:**
- Модель «Telegram — основной канал» (не дубль сайта)
- Эксклюзивность ассортимента как аргумент для покупки именно здесь
- Коллаборации для привлечения трафика в бота

**Ссылки:**  
- [Кейс на TGShop](https://tgshop.io/shops/casher-collection/)

## 5. Clothing Line (@clothing_line_bot)

**Бот:** @clothing_line_bot  
**Тип:** нативный Telegram Mini App  
**Ниша:** магазин одежды

**Что делает:**  
Полноценный Mini App (не бот-меню). Визуальный каталог с карточками товаров, работает как мобильное приложение внутри Telegram. Представлен в каталоге findmini.app.

**Сильные стороны:**
- Нативный Mini App — выглядит как приложение, а не как чат
- Визуальная сетка товаров с фото
- Можно открыть и пощупать прямо сейчас

**Слабые стороны:**
- Мало публичной информации о функционале
- Небольшая аудитория

**Что взять себе:**
- Формат нативного Mini App — именно к этому мы стремимся
- Визуальная подача каталога сеткой

**Ссылки:**  
- [Clothing Line на findmini.app](https://www.findmini.app/clothing_line_bot/)

## Главные выводы

1. **Конструкторы (TGShop) дают быстрый старт, но шаблонный вид.** Fashion Fabrique и CASHER выглядят одинаково. Наш кастомный Mini App — конкурентное преимущество по дизайну.

2. **Большинство не имеют вишлиста.** Только Yammy реализовал избранное. Это наша возможность выделиться.

3. **Бот-формат (G.Shop) устарел.** Сообщения в чате вместо визуального каталога — медленно и неудобно. Mini App — единственный правильный путь.

4. **Фото на моделях — стандарт.** Все успешные магазины используют lifestyle-фото, не flat lay.

5. **Telegram как основной канал (не дубль сайта) — работающая модель.** CASHER и Fashion Fabrique доказывают это результатами.
